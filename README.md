# 1. Introduction

The **E-Learning Management System (ELMS)** is a web-based application
designed to simplify online education. Students can register, enroll in
courses, access study materials, submit assignments, and take quizzes.
Instructors can create and manage courses, upload learning materials,
and monitor student performance. Administrators manage users, courses,
and the overall system.

# 2. Problem Statement

Many educational institutions still manage course materials and student
records manually. A centralized online system is needed where students,
instructors, and administrators can efficiently manage educational
activities.

# 3. Objectives

-   Secure login and authentication
-   Manage students, instructors, and courses
-   Student enrollment
-   Learning materials
-   Assignments and quizzes
-   Relational database management

# 4. Scope and Modules

## Authentication Module

-   Registration
-   Login
-   Role-Based Authentication

## Course Management Module

-   Create, Update, Delete Courses
-   Upload Study Materials

## Student Module

-   Enroll in Courses
-   View Materials
-   Submit Assignments
-   Attend Quizzes
-   View Progress

## Admin Module

-   Manage Users
-   Manage Courses
-   View Reports
